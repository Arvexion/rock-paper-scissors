# ROCK PAPER SCISSORS
### A basic rock paper scissors game against a computer with buttons as the user's choices.

## The skills I have demonstrated throughout completing this:
1. How to use global variables to keep track of things.
2. How to use methods such as `querySelectorAll` to reference multiple items in a container.
3. How to use the `forEach` code to efficiently assign an `addEventListener` to each item.
4. how to edit the HTML using javascript such as replacing it with who won each round.
5. How to effectively use functions and when.

**In essence, this has been an good practice for me to get to know the syntax of each and improve my understanding of how they work**
